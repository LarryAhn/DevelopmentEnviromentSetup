# Python3 Install on Ubuntu Linux

```
$ sudo apt-get update

$ sudo apt-get install gcc

$ sudo apt-get install zlibc zlib1g zlib1g-dev
$ sudo apt-get install libreadline6-dev libreadline6
$ sudo apt-get install libssl-dev 


https://www.python.org/downloads/release/python-{version}/
ex ) https://www.python.org/downloads/release/python-361/
$ wget <download link>

$ tar -zxvf Python-{version}.tgz

# in Python directory
$ ./configure
$ make
$ sudo make install
```
